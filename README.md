
Hubot nagios4
============

hubot script to interact with nagios4


Install
-------

In hubot project repo, run:

`npm install hubot-nagios4 --save`

Then add **hubot-nagios4** to your `external-scripts.json`:

```json
["hubot-nagios4"]
```

Commands
--------

`hubot nagios help` - Get help text
